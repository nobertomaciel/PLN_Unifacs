# Principais bibliotecas para uso em PLN

## Iniciante (conceitos básicos de PLN)
> * SciKit Learn – ML clássico para texto (TF-IDF, classificação, clustering): https://scikit-learn.org/stable/index.html
> * NLTK – fundamentos de tokenização, stemming, corpora e análise linguística: https://www.nltk.org/
> * TextBlob – API simples para sentimento, POS tagging e análise básica: https://textblob.readthedocs.io/

## Intermediário (PLN aplicado)
> * spaCy – pipelines eficientes para NLP moderno e aplicações reais: https://spacy.io/
> * gensim – modelagem de tópicos e embeddings semânticos: https://radimrehurek.com/gensim/
> * Polyglot – NLP multilíngue com suporte ao português: https://polyglot.readthedocs.io/
> * Stanza – análise linguística profunda baseada no Stanford NLP: https://stanfordnlp.github.io/stanza/

## Avançado (Deep Learning e IA Generativa)

> * Hugging Face Transformers – modelos de linguagem estado da arte: https://huggingface.co/transformers/
> * PyTorch – deep learning flexível para NLP avançado: https://pytorch.org/
> * TorchText – processamento de texto integrado ao PyTorch: https://pytorch.org/text/stable/index.html
> * TensorFlow – treinamento e deploy de modelos neurais de NLP: https://www.tensorflow.org/
> * TensorFlow Hub – modelos NLP pré-treinados prontos para uso: https://www.tensorflow.org/hub
